Type: Casual/Action

POWER SLEDS is a casual action game for the mobile marketplace. Set within a snowy forest the game capitalizes its fun on physics mechanics. The rules are simple; the player must guide a sled into objects that rest upon a suspended platform. The more objects the player overturns the larger the prize. The player must fill the score bar to advance into the next challenge.
