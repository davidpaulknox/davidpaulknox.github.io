SCRIPTING::
The level takes place within a cargo vessel, which is travelling between two battling start cruisers.

Additional Kismet scripts were needed to create the ensuing battle. I created twelve points (vectors) on each ship that the Kismet script would randomly select from. Once selected, the script would then spawn a projectile aimed at that point.

With both ships doing this in unison, the results were a cleverly orchestrate chaos between two heavy gunboats.
