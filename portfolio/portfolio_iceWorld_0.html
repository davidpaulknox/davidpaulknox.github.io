MAP OVERVIEW:
• Ice World is a capture-the-flag multiplayer map designed in the Unreal Tournament 3 engine. The map takes place within a snow-filled canyon just before sundown. This canyon contains two capture-the-flag bases, a central choke point, and a cave system that houses a dangerous leaking reactor. This reactor offers a risk/reward choice for players. The reactor path is shorter distance to the opposing team's flag, but damages those whom use it.
